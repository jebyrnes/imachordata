Sometimes, you just need to use a plyr

Jarrett Byrnes
Jul 10, 2009

I haven’t posted anything about R-nerdery in quite some time. But I have to pause for a moment, and sing the praises of a relatively new package that has made my life exponentially easier. The plyr package. R has the capability to apply a single function to a vector or list using apply or mapply, or their various derivatives. This returns another vector or list. This is great in principal, but in practice, with indexing, odd return objects, and difficulties in using multiple arguments, it can get out of hand for complex functions.
